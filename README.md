    <h2 id="tools">Tools</h2>
    <h3>Compress Image</h3>
    <p> This tool allows you to compress images to reduce their file size. The compression quality can be adjusted according to your needs. </p>
    <h4>How to Use</h4>
    <ol>
      <li>Run the <code>compfile.py</code> script.</li>
      <li>Enter the input folder path containing the images you want to compress.</li>
      <li>Enter the output folder path where the compressed images will be saved.</li>
      <li>Enter the compression quality (0-100).</li>
    </ol>
    <h3>Crop to Circle</h3>
    <p> This tool allows you to crop images into a circular shape, which can be useful for creating profile pictures. </p>
    <h4>How to Use</h4>
    <ol>
      <li>Run the <code>cropfile.py</code> script.</li>
      <li>Enter the input image file path.</li>
      <li>Enter the output image file path.</li>
    </ol>
    <h2 id="installation">Installation</h2>
    <p> To run the Photo Editing Tools, you will need to have Python installed on your machine. If you don't have Python installed, you can download it from the <a href="https://www.python.org/downloads/">official Python website</a>. Once Python is installed, you can run the tools using the following steps: </p>
    <ol>
      <li>Clone the repository or download the source code with the command: <code>git clone https://github.com/ryanshatch/Photo-Editing-Tools.git</code>. </li>
      <li>Navigate to the directory where you downloaded the programs. </li>
      <li>Install the required dependencies with the command: <code>pip install -r requirements.txt</code>. </li>
      <li>Run the desired tool with the command: <code>python compfile.py</code> or <code>python cropfile.py</code>. </li>
    </ol>
